The CSS Grid

The CSS Grid is a powerful tool that CSS gives us to easily align items and create two-dimensional layouts. Compared to Flexbox which can deal with either row or column (but not both at the same time), Grid can work on the row and the column at the same time.

What Is CSS Grid?

The Grid is a CSS model that provides a better way to create horizontal layouts and grids (like the grid layout in the homepage of YouTube). Grid is a two-dimensional layout method for arranging items in rows AND columns at the same time. In order to implement Grid, if we have an element (like a div) with two div children, we need to do two steps: 1) set the display property to grid, and 2) set the number of columns of the grid. To do the second step, we have several options but in this example we can set the grid-template-columns property to say 100px 100px.

(note: the widths are reduced to 80% on smaller screens)


<div class="container">
  <div class="item-1">1</div>
  <div class="item-2">2</div>
</div>



.container {
  display: grid;
  grid-template-columns: 100px 100px;
}
1
2

Now, even without setting the grid-template-columns, setting the display to grid alone will allow you to manipulate the element with other Grid properties. Take note that the numbers 1 and 2 inside the divs are in the center. This is because I have set the boxes themselves as grid containers as well. And then the place-items was also added in the CSS and given the value of center. 


.grid-items {
display: grid;
place-items: center;
}

Now, notice that the boxes have lined up on the left horizontally and took as much as the values of the grid-template-columns. Recall that divs are block items and takes the entire width. But when they are set into the Grid, they only take up as much columns instead.

Now, let's see what happens if we added a paragraph inside the box 2.


<div class="container">
  <div class="item-1">1</div>
  <div class="item-2">2<p>word and more words</p></div>
</div>

The result is that the fourth div "wraps" to the second row. If there are more columns in the html than what is set in the grid-template-columns, it will start to wrap and create new rows.

The Free Space (fr)

The Free Space or fr is a special value in grid that takes up the remaining amount of space on the grid. Looking at our very first example grid, let's change the grid-template-columns property to see the resulting grid.

1
2

.container {
  display: grid;
  grid-template-columns: 100px 1fr;
}
1
2

You can see that the first column is 100 px and the second div was changed in the grid-template-columns to 1fr and took up the remaining amount of space set by the grid parent. If we set the width of the parent to the entire width of the page, look what happens:

2

As you can see, the fr will take all the remaining space given by the parent div. Let's try to add one more div to see some properties of the fr and also add a column in the grid-template-columns.


.container {
  display: grid;
  grid-template-columns: 100px 1fr 1fr;
}
1

3

The second and the third div equally divided the remaining space. If we change the third column to 2fr:


.container {
  display: grid;
  grid-template-columns: 100px 1fr 2fr;
}
1
2
3

The 2st column took one-third of the space and the 3th column took up two-thirds of the space. So the fr is like a ratio.
