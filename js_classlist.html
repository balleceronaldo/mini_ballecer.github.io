The Basics of classList

classList is one of the most commonly used DOM property in JavaScript. One might think that classList is as simple as an add and remove method, but there is so much more to classList. In this article we will explore this add and remove method and more.

The Basics

When you access the classList of an element, you can use the add method to add a class and the remove method to remove a class. This is pretty straight forward, but most people don't know that you can pass as many classes as you want to these methods and they will add/remove them all. You can also access the string of all classes by using the value property. Check the code below in the developer tools to see the result. The following code is a format for doing this. 


element.classList.add('new-class', 'another-class')
console.log(element.classList.value)
// new-class another-class

element.classList.remove('another-class')
console.log(element.classList.value)
// new-class
Contains

You also need to check if an element has a specific class in JavaScript in order to perform a certain operation. This is luckily very easy to do with classList. You just call the contains method and pass the name of the class you want to check. This method will return true if the element has that class and false if it doesn't. You may open the developer tools console and paste the code below to check. (note: I added a div with a class of example) 


//run this code first first
example = document.querySelector(".example");
console.log
(example.classList.contains('new-class'))
// false

//run next
example.classList.add('new-class')
console.log
(example.classList.contains('new-class'))
// true
Toggle

My favorite method of classList is the toggle method. This method lets you toggle a class on/off depending on if the class is already on the element.

We can also use element selection by class name using getElementsByClassName. In our example, let's select the parents using this method.


element.classList.toggle('new-class')

The above is the same as the below.


if (element.classList.contains('new-class')) {
  element.classList.remove('new-class')
} else {
  element.classList.add('new-class')
}

This is not the only use of toggle, though. Sometimes you need to toggle a class based on a boolean value. For example if the boolean win is true then you want to add the win class to an element otherwise if win is false you want to remove the win class. Doing so is as simple as the below.


if (win) {
  element.classList.add('win')
} else {
  element.classList.remove('win')
}
