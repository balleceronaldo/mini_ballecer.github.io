JavaScript DOM Traversal

Being able to select and traverse the DOM (Document Object Model of the HTML) to manipulate it in JavaScript is one of the most important skill for a web developer. There are nine (9) techniques in order to master traversing or navigating the DOM. We will explore the first four of them.

Basic HTML and CSS

Below is the basic html and css code for our example. All divs are given the display property of flex. We have one grandparent element colored lightblue and which encloses all the elements. We have two lightpink parent elements with two lightgreen children inside each of them. 


<div class="grandparent">
  <div class="parent">
    <div class="child"></div>
    <div class="child"></div>
  </div>
  <div class="parent">
    <div class="child"></div>
    <div class="child"></div>
  </div>
</div>



div {
  display: flex;
  margin: .5rem;
  padding: 1rem;
  border: 3px solid black;
  min-height: 1rem;
  flex-grow: 1;
}

.grandparent {
  background-color: lightblue;
  flex-direction: column;
  margin-top: 30px;
}

.parent {
  background-color: lightpink;
}

.child {
  background-color: orangered;
}
Selecting by ID

The most simple way to traverse the DOM is by selecting the element from the document by using the ID selector. First, we must add and id to an element. Let's try to add and id to the grandparent. 


<div class="grandparent" id="granpa-id">
  <div class="parent">
    <div class="child"></div>
    <div class="child"></div>
  </div>
  <div class="parent">
    <div class="child"></div>
    <div class="child"></div>
  </div>
</div>

Now, let's edit the blank script .js file and add the following code and save the file to see the result:


const grandparent = document.getElementById("grandpa-id");

grandparent.style.backgroundColor = "#333";

We can see that the lightblue color of the grandparent has been replaced by the darkgray color because we selected the element by id using the getElementById method and then change the background-color property using the element.style.property = new style format.

We can also use element selection by class name using getElementsByClassName. In our example, let's select the parents using this method.


const parents = Array.from
(document.getElementsByClassName("parent"));

parents.forEach(changeColor);

function changeColor(element) {
  element.style.backgroundColor = "#333"
}

Take note that we have to add Array.from() because document.getElementsByClassName will not return an array but an HTML Collection which needs to be converted to an array. We also converted the element.style.backgroundColor = "#333" into a function we named as changeColor. This found 2 elements with the class of parents and changed them to the darkgray color (#333).

The querySelector and querySelectorAll

This is probably the most popular selector and can be used with almost all types of DOM navigation. The querySelector is used to select single elements while querySelectorAll is for multiple selection. Let's say we want to select the #grandpa-id again.


const grandparent = 
document.querySelector("#grandpa-id");

changeColor(grandparent)

function changeColor(element) {
  element.style.backgroundColor = "#333"
};

The querySelector is used like a normal CSS selector. We can select by tag names, id (by using the # sign), class (by using a period) and even pseudo classes (like button:hover). Let's try to select the grandparent class again.


const grandparent = 
document.querySelector(".grandparent");

changeColor(grandparent)

function changeColor(element) {
  element.style.backgroundColor = "#333"
};

If we use querySelector for the parent class, it will only select the first class it finds and then stop but there will be no error. Let's try to change the first parent (pink) to red.


const parent = 
document.querySelector(".parent");

changeColor(grandparent);

function changeColor(element) {
  element.style.backgroundColor = "red"
};

If we want to change the color of all the parents, then we use the querySelectorAll.


const parent = 
document.querySelectorAll(".parent");

parents.forEach(changeColor);

function changeColor(element) {
element.style.backgroundColor = "navy"
};
